# HDCryptorTools
OC 加密工具类



## Requirements

ios >= 8.0

## Installation

HDLAuthID is available through [CocoaPods](https://cocoapods.org/). To install it, simply add the following line to your Podfile:

```
pod 'HDCryptorTools'
```
